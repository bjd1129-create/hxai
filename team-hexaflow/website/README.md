# 大唐智能官网

> HXAI 华夏智能研究院官方网站
> 部署平台：Vercel
> 域名：hxai.vercel.app

---

## 🚀 快速部署

### 1. 安装依赖

```bash
npm install
```

### 2. 开发模式

```bash
npm run dev
```

访问：http://localhost:3000

### 3. 构建

```bash
npm run build
```

### 4. 部署到 Vercel

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署
vercel --prod
```
